# Nikita Glazkov — Portfolio

Personal portfolio website of Nikita Glazkov, Senior / Lead Android Engineer.

[View website](https://biggemott.github.io/)

## Tech stack

- Astro
- TypeScript
- Vanilla CSS
- Playwright

## Development

Requires Node.js 22.12+ and npm.

```bash
npm install
npm run dev
```

## Validation

```bash
npm run format:check
npm run check
npm run build
npm run visual:test
```

## Visual review

```bash
npm run visual:capture
```

Generated screenshots are stored in `artifacts/visual-review/` and ignored by Git.
